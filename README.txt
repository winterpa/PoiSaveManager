MANUALLY BEFORE USING THE SOFTWARE.

Backing up your saves:
1) Choose a backup folder location
2) Click on BACKUP GAME SAVES. These files will automatically copy your current saves to your backup folder with the date/time and slot
	number on them. 

Backing up individual saves:
1) Click on the Arrow next to each save slot.
2) Give your save file a name. 
3) Click ok.

Choosing a backup folder location:

1) Select a backup folder location. This is the location that you will be saving your backup saves too. Make this folder ahead of time.
2) A list of all saves available to be used as backups will be listed in the box on screen. 

Replacing a backup save:
1) Choose a backup folder location
2) Click on the save you would like to use
3) Click the "Set Save" button for the save you would like to replace (if you would like to place the first slot, hit "Set Save 1").
4) After you have replaced your saves, hit BACKUP SAVES button to backup your saves.
5) Click on Replace Game Saves to replace the current game saves with your backups.

Deleting a save from your backup folder:
1) Choose a backup folder location
2) Click on the save you wish to delete.
3) Click on Delete Save.

Renaming a backup save:
1) Choose a backup folder location
2) Click on Rename Save
3) Give your save file a name
4) Click ok.
5) Your save will now have the new save name.
